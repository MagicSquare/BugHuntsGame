# La chasse aux bugs #

## Dependencies

Dependencies of the web app:

    # Inside the root folder
    bower install

Dependencies of ionic/cordova (if used)

    # Inside the root folder
    cordova platform add browser
    cordova platform add android

## Development

Serve with ionic

    # Inside the root folder
    ionic serve

## Build & deploy

Build everything

    # Inside the root folder
    cordova build

### Android

    # Inside the root folder
    cordova build android
    cordova run android
